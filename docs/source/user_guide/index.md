# User Guide

The Biofilter 4 User Guide is intended for users who want to **install, configure,
and use Biofilter** to explore, annotate, filter, and model biological knowledge.

This guide focuses on **how to use Biofilter**, not how to extend or modify its
internal architecture. It covers common user workflows such as:

- Installing Biofilter
- Connecting to a knowledge base
- Creating and managing local databases
- Running built-in reports
- Exploring data through the query interface
- Executing ETL pipelines as a user
- Inspecting configuration and logs

Advanced topics related to extending Biofilter, writing new ETL packages,
or developing custom reports are covered in the **Developer Guide**.

If you are new to Biofilter, start with **Installation & Setup** and follow the
sections in order.
